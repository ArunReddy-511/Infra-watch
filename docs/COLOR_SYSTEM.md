# Color System and Design Tokens Documentation

## Introduction
This document outlines the color system and design tokens used in the Infra-watch project. Having a well-defined color system ensures consistency and enhances the user experience across the application.

## Color Palette

### Primary Colors
- **Primary Color**: `#007bff` (Blue)
- **Primary Light**: `#66b3ff`
- **Primary Dark**: `#0056b3`

### Secondary Colors
- **Secondary Color**: `#6c757d` (Gray)
- **Secondary Light**: `#b9bdc9`
- **Secondary Dark**: `#495057`

### Status Colors
- **Success**: `#28a745` (Green)
- **Danger**: `#dc3545` (Red)
- **Warning**: `#ffc107` (Yellow)
- **Info**: `#17a2b8` (Cyan)
- **Light**: `#f8f9fa` (Light Gray)
- **Dark**: `#343a40` (Dark Gray)

## Design Tokens
Design tokens are the visual design atoms of the project, helping us maintain consistency in our design across various platforms.

### Color Tokens
- **color-primary**: `#007bff`
- **color-secondary**: `#6c757d`
- **color-success**: `#28a745`
- **color-danger**: `#dc3545`
- **color-warning**: `#ffc107`
- **color-info**: `#17a2b8`
- **color-light**: `#f8f9fa`
- **color-dark**: `#343a40`

## Usage
Utilize the above color tokens in your stylesheets, ensuring that your designs remain in line with the defined color system. The tokens can be used within CSS, SCSS, or any other styling frameworks that support variable usage.

## Conclusion
Adhering to a cohesive color system and utilizing design tokens will help in building a more user-friendly interface and will allow us to manage our design language effectively. For any questions or suggestions regarding the color system, feel free to reach out!
