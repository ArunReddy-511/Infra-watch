# Setup and Installation Instructions

## Prerequisites
- Ensure you have Node.js (version 14 or higher) installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).
- Install Git for version control from [git-scm.com](https://git-scm.com/).

## Clone the Repository
First, clone the repository to your local machine:
```bash
git clone https://github.com/ArunReddy-511/Infra-watch.git
```

## Navigate to the Project Directory
```bash
cd Infra-watch
```

## Install Dependencies
Run the following command to install the necessary dependencies:
```bash
npm install
```

## Start the Application
You can start the application using:
```bash
npm start
```

## Running Tests
To run the tests, use:
```bash
npm test
```
